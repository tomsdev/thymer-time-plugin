# Thymer Time Command Plugin

A global AppPlugin that adds a Time command to the command palette to insert the current time.

## Installation

1. In Thymer, `Cmd+P` → "Plugins"
2. Create a new Global Plugin
3. Copy paste content of `plugin.json` into Configuration
4. Copy paste content of `plugin.js` into Custom Code

## Usage

1. `Cmd+P` → "Time"

## Known Limitations

The caret cursor position is not updated.
